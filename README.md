# hyperpop-2021



Проект с одной целью: ознакомить среднестатистического слушателя музыки с тем, что переживал русский гиперпоп в 2021 году.  

Сделано как небольшой визуальный черновик.



## Что внутри

- статическая страница (HTML/CSS/JS)

- медиа в `assets/`



## Запуск локально

Открой `index.html` в браузере.
